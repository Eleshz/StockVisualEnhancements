Installation Instructions:
1. Download the latest release of SVE (HR, MR, or LR version) from Spacedock
2. Extract StockVisualEnhancements folder into your Gamedata

**DO NOT INSTALL EVE SEPERATELY, EVERYTHING IS BUNDLED IN THIS DOWNLOAD** 


Known Issues:
- Zooming/flying past a certain altitude might cause a very rough visual transition in some cases.
- Sometimes the city day textures show up pitch black. No clue what is causing this :(
- General scatterer visual bugs. Blackrack is working on getting these issues fixed. Be patient
